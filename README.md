# Proreserv Rapport

Dateien:
- index.html
- app.js

## Start
Einfach index.html im Browser öffnen.

## GitHub Pages
Beide Dateien ins Repo hochladen und GitHub Pages aktivieren.

## Funktionen
- mehrere Einträge pro Tag
- Speichern ohne Export
- Bearbeiten / Löschen
- Lieferanten-Dropdown
- automatische Korrekturen
- Wochenrapport Excel-Export
- JSON Backup

Die Feldstruktur orientiert sich an deinem Beispielrapport mit Tag, Datum, Zeit, Minuten, Kilometer, Lieferant, Modul, Markt sowie getrennten Spesenfeldern.
